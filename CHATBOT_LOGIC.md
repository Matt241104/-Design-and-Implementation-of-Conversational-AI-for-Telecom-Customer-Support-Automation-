# Chatbot Logic

## Connectivity Issues
IF user says "no internet" OR "data not working"
→ Ask user to:
    - Restart device
    - Check data balance
    - Enable mobile data
→ IF not resolved
    → Escalate (create task)

---

## Billing Issues
IF user asks about billing
→ Provide billing explanation
→ IF complex
    → Escalate

---

## WiFi Issues
IF user says "WiFi not working"
→ Restart router
→ Check cables
→ IF not resolved
    → Escalate

---

## Escalation
IF chatbot cannot resolve
→ Collect user info
→ Create ClickUp task
→ Assign to support team
